This allows adding Title to the ListItem.
This, Receives all [Text](text#props) props.

## Usage

<div class="row inline-flex-center">
  <div class="col col--3">
    <h4>Import</h4>
  </div>
  <div class="col col--9">
    <CodeBlock>{`import { ListItem } from '@rneui/themed';`}</CodeBlock>
  </div>
  <div class="col col--3">
    <h4>
      Theme Key{" "}
      <a href="../customizing#using-themeprovider">
        <BiInfoCircle />
      </a>
    </h4>
  </div>

    <CodeBlock>{`ListItemTitle`}</CodeBlock>
  </div>
</div>

:::note
Includes all [Text](https://reactnative.dev/docs/text#props) props.
:::
