`DialogTitle` allows you to add title to the Dialog.

## Usage

<div class="row inline-flex-center">
  <div class="col col--3">
    <h4>Import</h4>
  </div>
  <div class="col col--9">
    <CodeBlock>{`import { Dialog } from '@rneui/themed';`}</CodeBlock>
  </div>
  <div class="col col--3">
    <h4>
      Theme Key{" "}
      <a href="../customizing#using-themeprovider">
        <BiInfoCircle />
      </a>
    </h4>
  </div>
  <div class="col col--9">
    <CodeBlock>{`DialogTitle`}</CodeBlock>
  </div>
</div>

## Props

<div class='table-responsive'>

| Name         | Type       | Default | Description                                 |
| ------------ | ---------- | ------- | ------------------------------------------- |
| `title`      | string     |         | Add Dialog title.                           |
| `titleProps` | TextProps  |         | Add additional props for Text component.    |
| `titleStyle` | Text Style |         | Add additional styling for title component. |

</div>
